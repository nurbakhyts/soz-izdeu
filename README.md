# Қазақша сөз іздеу

Сөздің басы, соңы, құрамы, ұзындығы бойынша қазақша сөз іздеу құралы.
`?` белгісі — кез келген бір әріп (үлгі бойынша іздеу).

**Сайт:** https://nurbakhyts.github.io/soz-izdeu/

## Сөздік

Сөздік қоры — [taem/hunspell-kk](https://github.com/taem/hunspell-kk) жобасының
орфографиялық қазақ тілі сөздігі (~54 000 сөз), OpenOffice Lingucomponent жобасы
аясында жасалған. Лицензиясы: GNU GPL 2.0+, GNU LGPL 2.1+, Mozilla MPL 1.1+.

Идея бастауы — [jarjan/qate.js](https://github.com/jarjan/qate.js) (MIT).

## Жариялау (GitHub Pages)

1. Осы репозиторийді GitHub-қа жүктеңіз
2. Settings → Pages → Source: «Deploy from a branch» → `main` → Save
3. Бірер минуттан кейін сайт `https://ЛОГИН.github.io/РЕПОЗИТОРИЙ/` адресінде ашылады
